v# Gwen Basconcillo – Portfolio Website

This is my personal portfolio website showcasing my projects, skills, certifications, and experience as an IT student.

## Live Website

https://gwenbasconcillo.vercel.app/

## Overview

The portfolio highlights my work in web development, UI/UX design, and system development. It includes interactive elements, a project showcase, and a contact form for communication.

## Features

* Responsive and modern design
* Dark and light mode toggle
* Interactive mini game (Stellar Dash)
* Projects and certifications section
* Contact form powered by Formspree
* Image preview modal for projects

## Technologies Used

* HTML
* CSS
* JavaScript
* Formspree
* Font Awesome
* Google Fonts
